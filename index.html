tambola

This is a online tambola playing web application.

Presently many users can play the same game but the notification of winner is not done

It is written to teach the basics of google appengine to students.

This project implements following from google app engine and objectify library

  • The folder structure of publishing apps on google app engine in Java.
  • How to save and retrieve data from the google datastore using the objectify library

For the frontend of the application I have used a very good responsive framework by Zurb known as Foundation
